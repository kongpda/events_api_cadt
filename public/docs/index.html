Introduction

Base URL: http://events_api.test

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Request      

POST api/events

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

title   string   

Must not be greater than 255 characters. Example: bkkrijcquvepkoima

slug   string   

Must not be greater than 255 characters. Example: zcyobt

description   string   

Must not be greater than 65535 characters. Example: Mollitia aspernatur consequatur voluptatem consectetur.

address   string   

Must not be greater than 255 characters. Example: htdvqopg

feature_image   string  optional  

Must not be greater than 255 characters. Example: hrsqxevlq

start_date   string   

Must be a valid date. Example: 2025-02-09T15:05:17

end_date   string   

Must be a valid date. Must be a date after start_date. Example: 2112-01-11

category_id   string   

The id of an existing record in the categories table. Example: tenetur

user_id   string   

The id of an existing record in the users table. Example: magnam

organizer_id   string   

The id of an existing record in the organizers table. Example: non

participation_type   string   

Example: paid

Must be one of:
  • paid
  • free
capacity   integer   

Must be at least 1. Example: 40

registration_deadline   string   

Must be a valid date. Must be a date before start_date. Example: 2013-09-24

registration_status   string   

Example: closed

Must be one of:
  • open
  • closed
  • full
event_type   string   

Example: hybrid

Must be one of:
  • in_person
  • online
  • hybrid
online_url   string  optional  

This field is required when event_type is online or hybrid. Must be a valid URL. Example: https://funk.com/incidunt-alias-est-excepturi-corporis.html

tags   string[]  optional  

The id of an existing record in the tags table.
